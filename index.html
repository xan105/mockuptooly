Sharpen

A simple convolution filter similar to the one photoshop uses after downscaling an image. Increase the value to sharpen the image.

Invert

0

This filter gradually inverts all the colors.

Brightness

100

Applies a linear multiplier making it appear more or less bright.

Contrast

100

Adjusts the contrast.
This filter controls the difference between dark and light parts.

Saturate

100

This filter controls the saturation of colors.
Supersaturated over 100% and remove all colors at 0%.

Grayscale

0

Also known as black and white.
This filter gradually converts all the colors to some shade of gray.

Hue rotate

0

This filter applies a hue-rotation to all the colors.

Opacity

100

This filter will apply transparency.
Opaque at 100% and fully transparent at 0%.

Blur

0

Applies a Gaussian blur. The value defines how many pixels on the screen blend into each other, so a larger value will create more blur.

Sepia

0

This filter gradually adds a sepia tinge like in some old photographs.

Magic Eraser is a great tool to quickly remove a background. You can use it to remove any color from your design but keep in mind that this tool was primarily designed to remove background as it will erase a very broad range of colors based on the approximate color that you picked. This behaviour is intended so that even with complicated backgrounds where there's colors overlapping or very close to each other, you just simply click on the background somewhere and it will vanish. There is no anti-alias like feature after the clip of the picked color, so your design may look out of place in some cases. Don't show this again Got it!
